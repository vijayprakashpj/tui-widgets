# Security Policy

## Supported Versions

Only the latest released version of this project is supported with security updates.

## Reporting a Vulnerability

Please report security vulnerabilities through GitHub private vulnerability reporting:
<https://github.com/ratatui/tui-widgets/security/advisories/new>.

Do not open a public issue for sensitive vulnerabilities. Public reports can make it harder to
coordinate a fix before affected users have an update path.
